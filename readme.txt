run ChipPy.py to start.

Press TAB to toggle console.
Console commands:
LOAD <ROM_NAME.CH8>  -  load rom from roms folder in directory
RES <VALUE>	     -  set resolution. Only provide width.
QUIT		     -  quit ChipPy
Any console command beginning with ERROR can be cleared by pressing enter.

Controls are mapped to:
	1234
	qwer
	asdf
	zxcv

Press space to enter debug mode. In debug mode you can step instructions with left arrow key.

All roms are made by David Winter.
